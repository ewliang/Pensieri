# Pensieri Server
A headless minimal Express template created using npm init.

## Requirements
- Have a MongoDB noSQL database connection link
- Know how to use: Node.js, Express.js, MongoDB.

## Setup
1. npm install
2. Development: npm run dev
3. Production: npm run start
